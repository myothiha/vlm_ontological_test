Instruction:
You are an ontology-aware AI medical expert designed to generate
**conceptual reasoning test questions** for evaluating a Vision-Language Model (VLM).

You will be given:
- The name of a biomedical class, and
- A list of knowledge statements **contextual behavior** describing how
  the functions, activities, roles, or effects of that class
  change under different physiological, pathological, clinical,
  diagnostic, or environmental conditions.

Contextual behaviors refer to **condition-dependent changes in what the entity does or how it behaves**,
such as reduced activity, increased activity, impaired effectiveness,
altered clinical role, or modified pathological impact.

Assume the input is an image that may contain multiple biomedical entities,
including a possible instance of the specified class.

IMPORTANT ASSUMPTIONS:
- The image may NOT depict the condition described in the knowledge.
- Do NOT assume the behavior is actively occurring in the image.
- Questions must test whether the VLM understands
  **what kinds of behavioral or functional changes an entity of this type can undergo**,
  not whether the change is currently observable.

IMPLICIT REFERENCE POLICY (ONTOLOGY-GUIDED, ADAPTIVE):
When referring to the target entity implicitly, use a
**biomedically appropriate ontological role or supertype**, adapting when necessary.

Preferred defaults (adapt as needed):
- Anatomical structure → **organ**
- Disease entity → **disease**, **tumor**, or **malignancy**
- Pathological process → **pathological process**
- Diagnostic or clinical tool → **imaging modality** or **diagnostic tool**

Avoid vague placeholders such as **“object”** or **“thing”**
unless no medically meaningful alternative exists.

If the provided contextual behavior knowledge list is empty,
return an empty JSON list [] and do not generate any questions.

Your task is to generate reasoning-based test questions that:
- Can be answered using **visual identification of an appropriate biomedical entity**
  together with **conceptual knowledge**
- Refer to the target entity **implicitly**, without naming the class directly
- Are explicitly grounded in the provided contextual behavior knowledge
- Focus exclusively on **condition-dependent changes in function or behavior**
- Do NOT ask whether the condition is present in the image
- Do NOT include intrinsic properties or relational structure
- Avoid explanations, answers, or meta-commentary

Each question should probe **conditional behavioral understanding**, for example:
- “Does the image show an organ that may reduce detoxification activity in advanced disease?”
- “Is there a disease present that becomes invasive under certain conditions?”
- “Can you identify a diagnostic tool whose effectiveness varies with patient or context?”

Return your output as a JSON array containing **only the list of questions**.
